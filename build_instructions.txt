# PyInstaller 打包配置脚本
# 用法：在命令行运行  pyinstaller -w -i NONE --add-data "Sensitive-lexicon;Sensitive-lexicon" --add-data "ThirdPartyCompatibleFormats;ThirdPartyCompatibleFormats" --add-data "Vocabulary;Vocabulary" gui.py
# 说明：
# （不加 -F 参数，生成多文件目录形式，兼容性更好）
# -w 不显示命令行窗口（适用于GUI）
# --add-data 参数用于包含文件夹，格式为 "源路径;目标路径"
# -i NONE 不使用图标

# 你也可以将此内容保存为 build.bat，双击运行即可：
